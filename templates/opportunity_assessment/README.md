# Opportunity Assessment templates

Place Excel template files for task-level Opportunity Assessment here.

## Current files

- `Detailed Scoping Document (task-level Opportunity Assessment).xlsx` — base template used when users click **Download template**. The API copies this file, pre-fills `migration_request_id`, and injects L1→L2→L3→L4 cascading dropdowns from `fpo_mapping`.

Additional related documents can be added to this folder later.
